Description

Once we have the layout, let's continue with our input. So far, the buttons are plain pictures. They don't seem to work. Infuse life into them by adding operations and showing the numbers you've typed on the screen.

Objectives

Power up the buttons you have created in Stage 1.

  1. After clicking on any button, a corresponding number should be added to EditText. Make sure you do not overwrite the number you've typed in earlier.
  2. Users are not allowed to type multiple 0 when EditText contains a single 0. The same thing should apply to the dot button (only one dot can be displayed).
  3. Make the Clear button set EditText back to 0.
