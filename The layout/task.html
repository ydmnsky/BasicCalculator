Description

A calculator is a very useful app for daily use, so let's start designing our own one! Your app will handle basic mathematical operations such as addition, subtraction, multiplication, and division.
Users must be able to input numbers by pressing big enumerated buttons on the screen, choose the operation, and then see the results of their calculations.

Objectives

Add the following components to the application layout:

  1. An EditText field where users can see the number they type and the results. Set this EditText field to "" (an empty string) by default. The field should not be editable, and the EditText hint should be set to 0;
  2. Several different buttons. Add buttons from 0 to 9, a dot ., add +, subtract -, multiply *, divide /, equal sign =, and clear buttons. Use the following ID's:
  • button0, button1, button2 , and so on for numbers;
  • dotButton;
  • equalButton;
  • clearButton;
  • addButton;
  • subtractButton;
  • multiplyButton;
  • divideButton.

Please, enumerate the number buttons from 0 to 9. For other buttons, you are free to use any text or symbol you want.
